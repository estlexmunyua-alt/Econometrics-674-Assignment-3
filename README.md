# Econometrics 674 Assignment 3 - Texas Housing VAR Analysis

## Project Overview
Vector Autoregressive (VAR) model analysis of Texas housing market data featuring:
- VAR(8) model implementation
- 12-month forecasting
- Structural break detection
- Robust error handling

- VAR modeling
